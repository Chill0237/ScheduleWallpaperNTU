# 台大課表桌布產生器 (NTU Course Wallpaper Generator)

這是一個專為台大課程網設計的 Chrome 擴充功能，可以將你的選課結果一鍵轉換為手機桌布。

## 主要功能
* **自動抓取**：直接讀取台大課程網的「選課結果」頁面，不會存取任何個人資料。
* **高度客製**：可調整顏色、字體顏色、字體大小、課程自訂名稱。
* **版面微調**：支援調整上下左右留白、文字對齊方向 (左/中/右)。

## 🚀 如何安裝 (30秒完成)
由於本工具未上架 Chrome 線上商店，請依照以下步驟手動安裝：

1.  **下載**：點擊 [這裡下載壓縮檔] (請換成你的 Google Drive 或 GitHub 連結) 並 **解壓縮** 到電腦某個資料夾。
2.  **開啟擴充功能管理**：
    * 在 Chrome 網址列輸入 `chrome://extensions/` 並按下 Enter。
    * 或者點選瀏覽器右上角「拼圖圖示」 -> 「管理擴充功能」。
3.  **開啟開發模式**：
    * 打開右上角的 **「開發人員模式」** 開關。
4.  **載入**：
    * 點選左上角的 **「載入未封裝項目 (Load unpacked)」**。
    * 選擇剛剛 **解壓縮出來的資料夾**。
5.  **完成**：現在你可以在擴充功能列看到它了！

## 📖 使用教學
1.  進入 [台大課程網](https://course.ntu.edu.tw/) 並登入。
2.  前往 **「查詢選課結果」** 頁面。
3.  點擊瀏覽器右上角的擴充功能圖示。
4.  調整你喜歡的樣式與配色。
5.  點擊 **「下載圖片」** 即可獲得桌布！
