# NTU 課表桌布產生器 (NTU Course Wallpaper Generator)

這是一個專為國立臺灣大學 (NTU) 學生設計的 Chrome 擴充功能，能夠將課程網的選課結果轉換為美觀、可高度客製化的手機桌布。

## 主要功能

* **隱私安全**：所有資料處理完全在您的瀏覽器端完成，**絕對不會**上傳任何個人課表資訊或學號密碼。
* **自動抓取**：讀取台大課程網的選課結果（初選、加退選及最終結果），自動偵測並解析課程。
* **高度客製化**：
    * **排版調整**：自由調整上下左右留白、圓角大小、文字對齊方向。
    * **顯示設定**：可開關課名、教室、星期、節次時間的顯示，並微調字體大小與位置。
    * **內容編輯**：支援直接修改課程名稱與教室名稱。
* **豐富主題**：內建 8 種配色主題（粉彩、冷色、暗黑、莫蘭迪、大自然、復古、霓虹、咖啡），每種主題包含 15 組配色，並支援隨機配色不撞色。
* **完整控制**：支援離線預覽與空課表顯示。

## 安裝方式

1.  前往 [Chrome Web Store 頁面] (https://chromewebstore.google.com/detail/ntu%E8%AA%B2%E8%A1%A8%E8%BD%89%E6%A1%8C%E5%B8%83/ecgkdpafjhjamknmjemhghhgdopinmjc)。
2.  點擊 **「加到 Chrome」** 即可開始使用。

## 使用教學

1.  進入 [台大課程網](https://course.ntu.edu.tw/) 並登入。
2.  前往 **選課結果** 頁面（支援初選、加退選、最終結果），點擊課表。
3.  點擊瀏覽器工具列上的擴充功能圖示。
4.  **排版設定**：調整版面佈局、字體大小與顯示項目。
5.  **顏色設定**：選擇喜歡的主題風格，或手動修改全域背景與字體顏色。
6.  點擊 **下載圖片**，將製作好的課表儲存為 PNG 圖檔。

## 系統需求與相容性

本擴充功能基於 **Manifest V3** 開發，原生支援所有 **Chromium 核心** 的桌面版瀏覽器：

* **Google Chrome** (推薦)
* **Microsoft Edge**
* **Brave** / **Vivaldi** / **Opera** / **Arc**

**不支援：**
* 手機版瀏覽器 (iOS/Android)
* Safari (macOS)
* Firefox (尚未進行適配測試)

**作業系統：**
* 支援 Windows, macOS, Linux (只要能安裝上述瀏覽器即可)。
